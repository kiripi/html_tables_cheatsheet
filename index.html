HTML Table Tags

Tag Name Description
<table> Table Defines a table
<thead> Table Head Used to group header content
<tbody> Table Body Used to group the body content
<tfoot> Table Foot Used to group footer content
<th> Table Header Defines a header cell
<tr> Table Row Defines a row
<td> Table Data A standard data cell

Table Attributes

Tag Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a <td> element should span
